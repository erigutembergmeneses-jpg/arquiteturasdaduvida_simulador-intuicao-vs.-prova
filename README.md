# Simulador "Intuição vs. Prova": O Caso do Relógio Roubado

![QR Code Simbólico](https://img.icons8.com/ios/452/qr-code.png)

Este é um simulador interativo que coloca o usuário no papel de um delegado que precisa decidir entre **confiar em sua intuição** (baseada no comportamento nervoso do suspeito) ou **confiar na prova** (um vídeo que estabelece um álibi).

O objetivo é expor a **ilusão de competência** na detecção de mentiras, conforme estudos de Elaad (2022), e refletir sobre vieses em decisões judiciais, usando exemplos de casos reais brasileiros.

## 🧠 Conceito

- O suspeito **se comporta de forma nervosa**: hesita, desvia o olhar, voz trêmula.
- O **álibi em vídeo** mostra o suspeito no saguão do hotel no horário do crime.
- O usuário deve escolher: **Intuição** ou **Prova**?
- O feedback mostra as consequências da escolha com base em pesquisas e casos reais.
